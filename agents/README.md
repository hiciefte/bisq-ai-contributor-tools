# Bisq Agents

This directory contains specialized AI agents with Bisq domain expertise.

## Agent Structure

Each agent is defined with specific expertise and behavioral patterns for Bisq development contexts.

## Planned Agents

### Trade Protocol Agent
Expert in analyzing Bisq's trade protocol state machines, identifying issues in trade flows, and suggesting improvements.

### P2P Networking Agent
Specialized in debugging P2P networking issues, analyzing network topology, and optimizing connection handling.

### Bitcoin Transaction Agent
Expert in reviewing Bitcoin transaction construction, fee estimation, and multi-signature wallet operations.

### DAO Governance Agent
Specialized in evaluating DAO proposals, analyzing voting mechanisms, and understanding governance processes.

### Security Audit Agent
Expert in identifying security vulnerabilities specific to financial P2P applications and cryptocurrency integration.
